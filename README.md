# Welcom to my lotto analysis

## What it does?

Gets all lottery results from [here](redeloteria.com.br) and plots graphs

## How to run

### Install requirements:

```
pip3 install -r requirements.txt
```

then

```
python3 ds.py
# or
py ds.py
```
